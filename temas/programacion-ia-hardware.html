Objetivo General

Incorporar conceptos de programación e inteligencia artificial en hardware de forma progresiva y práctica.

Clase 9: Primeros pasos con Python

Lógica y estructuras básicas

Se introduce Python desde la lógica de programación básica: variables, condicionales y bucles. El foco está en desarrollar pensamiento computacional desde problemas concretos.

Actividades sugeridas:

  • Adivina el número (juego con if)
  • Contador con for
  • Conversor de unidades

Objetivo: que los estudiantes comprendan la lógica antes de aplicarla a hardware o IA.

Clase 10: Python aplicado a datos e IA

Listas y funciones

Se trabajan estructuras de datos simples (listas, diccionarios), y funciones propias. Se introduce el concepto de entrada/salida de datos y manipulación básica de información.

Actividades sugeridas:

  • Crear un programa que almacene datos de sensores simulados (ej. temperaturas aleatorias) y calcule promedios.
  • Breve exploración de cómo una IA podría "predecir" el siguiente dato (introducción a patrones).

Opcional: mostrar cómo Python puede simular sensores para desarrollar lógica sin hardware.

Clase 11: Introducción a la IA desde Python

Decisiones inteligentes

Se introduce el concepto de IA como sistema que toma decisiones basadas en datos. Se usan ejemplos simples con lógica condicional y estructuras que simulan inteligencia.

Actividades sugeridas:

  • Crear un "asistente inteligente" en consola que recomienda ropa según el clima (dato ingresado por usuario).
  • Implementar un mini-clasificador basado en reglas (if/else) para identificar si una fruta es "dulce o ácida" según características.

Complemento: comparar luego con una pequeña red entrenada o un modelo en Teachable Machine para mostrar la diferencia entre IA "programada" y IA "aprendida".

Clase 12: Programación colaborativa

VS Code + Live Share

Introducción a herramientas modernas de desarrollo colaborativo, tanto remotas como presenciales.

Actividad sugerida:

En parejas, modificar un mismo archivo en tiempo real para resolver un desafío de codificación.

Clase 15: Mini Hackathon II

"Mi asistente inteligente en Python"

Se realiza una competencia amistosa en la que los grupos diseñan, programan y presentan un asistente virtual inteligente utilizando Python. Puede tratarse de un chatbot, un recomendador, o una herramienta que tome decisiones con base en datos, reglas o un modelo simple de IA.

Objetivos:

  • Aplicar conocimientos de lógica, estructuras de datos y funciones
  • Integrar conceptos de IA (clasificación, predicción, reglas inteligentes)
  • Fomentar la creatividad, el trabajo en equipo y la exposición de ideas

Desafío en equipos (2-4 personas):

Crear un programa en Python que cumpla al menos una de estas funciones:

  • Recomiende algo útil según datos (ropa, comida, rutas)
  • Clasifique datos con IA entrenada (como frutas, clima, emociones)
  • Simule un asistente conversacional básico con respuestas "inteligentes"

Requisitos:

  • Uso de funciones propias
  • Entrada de datos del usuario
  • Algoritmo de decisión (condicionales, listas, modelo IA básico o árbol de decisión)
  • Presentación del proyecto al resto de la clase (consola o visual simple)

Ejemplos posibles:

  • Asistente para elegir actividad según clima y estado de ánimo
  • Clasificador de canciones por estilo según palabras clave
  • Bot que ayuda a organizar una mochila escolar según materias del día

Cierre:

  • Cada equipo presenta su proyecto brevemente
  • Se valoran: claridad de código, funcionalidad, uso de IA o lógica avanzada, y creatividad
  • El facilitador puede dar menciones a "Mejor uso de IA", "Más útil", "Más creativo"
