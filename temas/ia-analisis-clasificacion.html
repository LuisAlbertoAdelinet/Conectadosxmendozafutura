Objetivo

Que los estudiantes comprendan cómo funcionan los modelos de IA básicos para analizar datos y clasificar información, utilizando lógica, listas, funciones y estructuras condicionales en Python, apoyados eventualmente por el modelo de IA local.

Fundamentos del aprendizaje

  • Clasificar es uno de los usos más frecuentes y útiles de la IA (imágenes, textos, personas, situaciones).
  • A nivel escolar se puede simular este proceso usando estructuras simples de Python: listas, condicionales, y funciones.
  • Se trata de pensar como una IA: ¿Qué datos tengo?, ¿Qué reglas uso para decidir?, ¿Cómo evalúo?

Contenidos clave

¿Qué es clasificar?

Agrupar o etiquetar elementos según criterios.

Ejemplos:

  • Correos spam/no spam
  • Temperatura alta/media/baja
  • Sentimientos positivos/negativos

Datos de entrada

Información que analizamos (listas, números, palabras).

Criterios de decisión

Reglas que definimos para etiquetar (por ejemplo: si la edad > 60 → "mayor").

Salida esperada

Etiqueta, grupo, respuesta ("apto", "revisar", "descartar").

IA real vs. IA simulada

  • No necesitamos entrenar modelos reales para aprender la lógica.
  • Podemos simular decisiones inteligentes con código tradicional + IA local como apoyo.

Ejercicios sugeridos

Nivel básico – Clasificadores por reglas simples

Clasificador de edades

  • Entrada: lista de edades
  • Salida: lista de etiquetas ("niño", "adolescente", "adulto")
def clasificar_edad(edad):
    if edad < 13:
        return "niño"
    elif edad < 18:
        return "adolescente"
    else:
        return "adulto"

Clasificador de temperaturas

  • Entrada: temperatura en grados
  • Criterios: baja < 15, normal 15–30, alta > 30

Nivel intermedio – Clasificadores basados en múltiples criterios

Clasificador de rendimiento académico

Datos: lista de promedios

Criterios:

  • ≥ 8: excelente
  • 6–7.99: aceptable
  • < 6: en riesgo

Clasificador de clima

  • Entrada: diccionario con temperatura y humedad
  • Condiciones anidadas:
    • Si temp > 30 y humedad > 70 → "calor húmedo"
    • Si temp < 10 → "frío"
    • Si humedad < 30 → "seco"

Nivel avanzado – Análisis + Clasificación con funciones más complejas

Clasificador de textos básicos

  • Entrada: lista de frases
  • Clasificar si contiene palabras clave positivas o negativas
def clasificar_sentimiento(frase):
    positivas = ["feliz", "genial", "me gusta"]
    negativas = ["malo", "odio", "triste"]
    for palabra in positivas:
        if palabra in frase:
            return "positivo"
    for palabra in negativas:
        if palabra in frase:
            return "negativo"
    return "neutral"

Clasificador personalizado por contexto

Los estudiantes eligen un tema:

  • Productos de kiosco: saludable/no saludable
  • Redes sociales: útil/perjudicial
  • Palabras: formales/informales

Actividad con IA local: "¿Cómo clasificaría esto la IA?"

Paso a paso:

  1. El grupo crea su propio clasificador por reglas en Python.
  2. Generan una lista de ejemplos (entradas posibles).
  3. Piden a la IA que también los clasifique:

    "Tengo una lista de frases. ¿Podés clasificar cada una como positiva o negativa?"

  4. Comparan la clasificación humana vs. la de la IA:
    • ¿Coinciden?
    • ¿En qué casos difieren?
    • ¿Por qué creen que pasa eso?

👉 Esta actividad estimula:

  • Pensamiento crítico
  • Mejora de algoritmos propios
  • Evaluación de sesgos

Estrategia por niveles

Nivel Actividades clave
Básico Clasificación con if simples sobre datos numéricos o strings
Intermedio Múltiples criterios, funciones con entrada/salida
Avanzado Procesamiento de texto, clasificación simulada vs. IA real

Proyecto final sugerido

"Mi clasificador inteligente"

Tema libre por grupo (cosas del aula, redes, hobbies, alimentos, etc.)

Etapas:

  • Elegir qué quieren clasificar
  • Definir criterios y etiquetas
  • Implementar con Python
  • Probar con casos reales
  • Comparar con la IA
  • Reflexionar sobre los resultados

🎯 Se puede presentar con una pequeña interfaz en consola, o como formulario en Google si se amplía.

Recursos útiles

  • Python básico: condicionales, listas, funciones
  • Modelos locales ligeros para preguntas de texto
  • Dataset simple generado por los estudiantes
  • Actividades desconectadas con lápiz y papel: clasificar tarjetas, organizar en grupos, hacer simulación humana.
