Objetivo general

Que los estudiantes diseñen y programen un asistente digital sencillo en Python, que simule una IA capaz de interactuar, responder preguntas básicas y ejecutar funciones útiles según criterios definidos por el grupo.

¿Por qué un asistente?

  • Representa un uso cercano de la IA, similar a asistentes como Alexa, Siri o ChatGPT.
  • Permite integrar conocimientos técnicos (funciones, condicionales, entrada/salida de datos) con habilidades blandas (empatía, diseño de experiencias, resolución de problemas).
  • Estimula la creatividad y la resolución colaborativa en torno a un producto con valor.

Componentes del proyecto

El asistente debe:

  • Leer una entrada del usuario (texto simple por consola).
  • Procesar el mensaje y decidir una respuesta según las palabras clave.
  • Realizar una acción útil o responder adecuadamente.

Etapas del desarrollo

1. Ideación del asistente (Design Thinking inicial)

¿Qué problemas resuelve? ¿Qué personalidad tiene? ¿Cómo ayuda?

Ejemplos:

  • Un asistente para estudiantes que recuerda tareas y explica temas.
  • Uno que da consejos de estudio o frases motivadoras.
  • Uno que simula responder preguntas sobre tecnología, clima, salud, etc.

2. Diseño de funcionalidades mínimas (MVP)

Entrada esperada Respuesta o acción del asistente
"¿Qué día es hoy?" Devuelve la fecha actual
"¿Qué me recomendás para estudiar?" Devuelve una sugerencia de contenido
"Estoy triste" Responde con una frase de ánimo
"Contame un chiste" Devuelve una frase graciosa

👉 Esto se puede codificar con estructuras if, listas de frases, funciones como datetime, etc.

3. Programación por módulos

  • procesar_entrada() → interpreta qué quiere el usuario.
  • responder() → selecciona la respuesta adecuada.
  • Funciones auxiliares para tareas como mostrar la hora, dar consejos, saludar.
def procesar_entrada(entrada):
    entrada = entrada.lower()
    if "hora" in entrada:
        return mostrar_hora()
    elif "triste" in entrada:
        return "¡Ánimo! Todo mejora con práctica 😊"
    elif "tarea" in entrada:
        return "Recordá que mañana hay que repasar Python."
    else:
        return "No entendí bien. ¿Podés repetirlo?"

4. Interacción con la IA local

Los estudiantes pueden consultar a la IA para:

  • Generar ejemplos de respuestas.
  • Obtener ayuda para escribir funciones.
  • Pedir ideas de nuevas funcionalidades.

Ejemplo de prompt útil para IA:

"Quiero que mi asistente en Python responda a frases de ánimo. ¿Podés darme una función que elija una frase al azar de una lista?"

5. Testeo y presentación

  • Simular conversaciones.
  • Recibir feedback de otros equipos.
  • Mejorar el asistente (iteración).
  • Presentar el producto en una breve demo.

🎤 Pitch final:

"Nuestro asistente se llama X. Ayuda a los estudiantes a Y. Lo hicimos con Python usando funciones, listas y condicionales. Lo que más nos costó fue Z, pero lo resolvimos así..."
