Objetivo de la clase

Que los estudiantes aprendan a trabajar en equipo sobre un mismo proyecto de código, compartiendo en tiempo real ideas, archivos y soluciones, utilizando Visual Studio Code y la extensión Live Share como entorno de trabajo colaborativo.

Contenidos clave

1. ¿Qué es la programación colaborativa?

  • Es una forma de escribir código en conjunto con otras personas, en tiempo real o de forma asincrónica
  • Permite resolver problemas más rápido, revisar código, y aprender entre pares
  • Fomenta habilidades blandas como comunicación, empatía, liderazgo y organización

2. Visual Studio Code (VS Code)

  • Editor de código liviano, gratuito, multiplataforma
  • Extensible con plugins como Python, Live Share, Jupyter, etc.

3. Live Share

Extensión de VS Code que permite compartir un proyecto en tiempo real.

Permite:

  • Que varios usuarios editen el mismo archivo
  • Compartir un servidor o terminal
  • Hacer sesiones de chat y comentarios en el código

4. Buenas prácticas al colaborar

  • Asignar roles (navegador, conductor, revisor)
  • Nombrar bien archivos y funciones
  • Escribir comentarios útiles
  • Hacer pausas para discutir el avance
  • Respetar los turnos de escritura

Ejercicios prácticos

Nivel básico

Exploración guiada

  • Dos estudiantes se conectan en la misma sesión Live Share
  • Uno escribe un código simple, el otro lo modifica
  • Observan la actualización en tiempo real

Corrigiendo juntos

  • Un código con errores simples
  • Un estudiante detecta, otro corrige
  • Diálogo sobre la solución

Nivel intermedio

Mini proyecto compartido

  • Crear juntos un programa que simule un menú
  • Dividir tareas: uno las opciones, otro la interfaz

Pair programming

  • Un estudiante dicta la lógica, otro escribe
  • Intercambiar roles periódicamente

Nivel avanzado

Refactorización colaborativa

  • Mejorar un código "sucio" en equipo
  • Modularizar y documentar juntos

Code Review

  • Un estudiante propone una solución
  • Otro revisa y sugiere mejoras

Integración con IA

¿Qué opina la IA sobre nuestro código?

Después de escribir juntos un código, los estudiantes pueden:

Ejemplo de interacción:

  1. Resumir su código en lenguaje natural
  2. Pedir sugerencias al modelo de IA BlackBox o Copilot
  3. Analizar las recomendaciones en equipo
  4. Implementar mejoras colaborativamente

Recursos complementarios

  • VS Code para estudiantes (offline-friendly)
  • Extensión Live Share (descarga directa)
  • Extensión BlackBox AI (descarga directa)
  • Extensión Copilot AI (descarga directa)
  • Instrucciones para entornos sin conexión: instalar extensiones manualmente .vsix

Proyecto final: "Creemos una solución juntos"

Cada grupo debe:

  • Plantear un problema real de su entorno
  • Dividir la solución en partes lógicas
  • Usar Live Share para programar simultáneamente
  • Documentar el trabajo y los roles
  • Opcional: comparar su solución con IA
