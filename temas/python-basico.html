Teoría: Lógica y estructuras básicas en Python

Objetivo: Que los y las estudiantes comprendan cómo estructurar un pensamiento lógico y lo plasmen en código Python utilizando variables, condicionales, bucles y funciones simples.

Contenidos clave

Variables y tipos de datos

  • int, float, str, bool
  • Entrada (input) y salida (print)

Operadores

  • Aritméticos (+, -, *, /, %)
  • Comparación (==, !=, >, <, >=, <=)
  • Lógicos (and, or, not)

Condicionales 

if condición:
    # bloque de código
elif otra_condición:
    # otro bloque
else:
    # bloque alternativo

Bucles

  • while: ejecuta mientras la condición sea verdadera
  • for: recorre una secuencia

Funciones simples

Definición con def, uso de parámetros y retorno (return)

Ejercicios sugeridos

Nivel básico

  • Calculadora simple

    • Pedir dos números y una operación (+, -, *, /)
    • Mostrar el resultado usando if

  • Par o impar

    • Leer un número y decir si es par o impar

  • Adivina el número

    • Número fijo (por ejemplo, 7)
    • El usuario intenta adivinarlo, con pistas de "mayor" o "menor"

  • Contador

    • Usar while para contar del 1 al 10

Nivel intermedio

  • Menú interactivo

    • Crear un menú con print e input, y que realice distintas funciones (como una mini app)

  • Conversor de unidades

    • Km a millas, °C a °F, etc.

  • Tabla de multiplicar

    • Usar for para mostrar la tabla del número que el usuario elija

  • Funciones reutilizables

    • Crear una función que reciba dos números y devuelva la suma

Nivel avanzado (opcional)

  • Generador de contraseñas simples

    • Usar random para generar cadenas con letras y números

  • Simulador de chatbot con IA local

    def preguntar_al_modelo(pregunta):
    # Enviar la pregunta al modelo local y devolver respuesta
    return respuesta_simulada

Sugerencia de integración con IA

Actividad: "El código me explica el código"

Los y las estudiantes escriben un pequeño script Python (ej: la calculadora o el adivinador).

Luego, lo copian en el sistema local con IA y le piden: "¿Podés explicarme cómo funciona este código paso a paso?"

El modelo devuelve una explicación textual, que ellos pueden contrastar con lo que aprendieron.

Esto estimula:

  • La autonomía
  • El pensamiento reflexivo
  • El aprendizaje colaborativo si se comparte entre pares

Extra: Trabajo grupal o por niveles

  • Grupo básico:

    Usa plantillas de código con espacios a completar

  • Grupo intermedio:

    Resuelve ejercicios con ayuda del modelo

  • Grupo avanzado:

    Crea un script útil para el aula o compañeros (como un generador de ejercicios, un quiz automático, etc.)
