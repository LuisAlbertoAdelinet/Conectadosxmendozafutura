Objetivo de la clase

Que los estudiantes aprendan a almacenar y procesar datos en estructuras de listas, y a modular el código mediante funciones reutilizables, aplicando estas herramientas a contextos simples de análisis de datos e inteligencia artificial.

Contenidos clave

1. Listas en Python 

# Creación
mi_lista = [1, 2, 3]

# Acceso por índice
primer_elemento = mi_lista[0]

# Métodos principales
mi_lista.append(4)    # Agregar
mi_lista.remove(2)    # Eliminar valor
mi_lista.pop()       # Eliminar último
mi_lista.sort()      # Ordenar
len(mi_lista)        # Longitud

2. Bucles sobre listas 

# Recorrer elementos
for elemento in lista:
    print(elemento)

# Recorrer con índices
for i in range(len(lista)):
    print(f"Posición {i}: {lista[i]}")

3. Funciones 

def calcular_promedio(numeros):
    return sum(numeros) / len(numeros)

def filtrar_mayores(lista, umbral):
    return [x for x in lista if x > umbral]

4. Datos y estructuras anidadas 

# Listas de listas
matriz = [[1, 2], [3, 4]]

# Diccionario simple
alumno = {
    "nombre": "Ana",
    "notas": [8, 9, 7]
}

Ejercicios prácticos

Nivel básico

Notas de un estudiante

  • Crear una lista con 5 notas
  • Calcular promedio usando sum() y len()
  • Mostrar la nota más alta y más baja

Lista de compras

  • Agregar productos con input() y .append()
  • Mostrar la lista completa

Nivel intermedio

Análisis de temperaturas

  • Ingresar 7 temperaturas y mostrar las 3 más altas
  • Calcular temperatura promedio por día

Visualización simple

  • Crear gráfico de barras con asteriscos
  • Ejemplo: print("*" * temperatura)

Nivel avanzado

Análisis de datos

  • Crear funciones de análisis estadístico
  • Generar insights con ayuda de la IA
def analizar_datos(datos):
    return {
        "promedio": sum(datos) / len(datos),
        "maximo": max(datos),
        "minimo": min(datos)
    }

Integración con IA

El asistente de análisis

Usando el sistema local de IA, podemos analizar nuestros datos y obtener insights valiosos:

Ejemplo de interacción:

# Datos para analizar
datos = [23, 45, 67, 12, 34]

# Preguntas para el modelo
prompt1 = f"¿Qué patrones ves en estos datos: {datos}?"
prompt2 = "¿Qué otros análisis sugerís hacer?"

Ideas de proyecto

"Encuesta en el aula"

Recolectar datos reales (edad, opinión, gustos), almacenarlos en listas, y analizar con funciones propias y comentarios del modelo local.

"Gestor de tareas"

Lista de tareas con opciones para agregar, eliminar, marcar como hecha (ideal para aplicar funciones + lógica).
