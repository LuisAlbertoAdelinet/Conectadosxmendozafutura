Objetivo de la clase

Que los estudiantes comprendan qué es una "decisión inteligente" en el contexto de la programación, cómo se relaciona con el concepto de inteligencia artificial, y que puedan simularla usando estructuras condicionales, funciones y estructuras de datos.

Contenidos clave

1. ¿Qué es una decisión inteligente?

  • Es una elección basada en datos y reglas, no al azar
  • Se usa en IA para predecir, clasificar, recomendar o reaccionar ante situaciones

2. Simulación de IA con lógica básica 

def recomendar_actividad(clima, energia):
    if clima == "lluvia":
        if energia > 7:
            return "Juegos de mesa en casa"
        else:
            return "Película y descanso"
    elif clima == "sol":
        if energia > 5:
            return "Salir al parque"
        else:
            return "Leer en el jardín"

3. Estrategias sencillas de IA

  • Clasificación por reglas (sistemas expertos)
  • Decisiones basadas en puntuaciones
  • Uso de estructuras de datos para conocimiento

4. Relación con la IA real

Si bien el aprendizaje automático es diferente a las decisiones por reglas, los conceptos básicos son similares. En este nivel simulamos la IA con lógica programada, mientras que en modelos más complejos las decisiones se entrenan.

            return "Animación familiar"

Nivel avanzado

Mini sistema de diagnóstico

# Base de conocimiento
enfermedades = {
    "gripe": ["fiebre", "tos", "dolor de cabeza"],
    "alergia": ["estornudos", "picazón", "congestión"],
    "cansancio": ["fatiga", "dolor muscular", "sueño"]
}

def diagnosticar(sintomas):
    resultados = {}
    for enfermedad, sint_esp in enfermedades.items():
        coincidencias = len([s for s in sintomas if s in sint_esp])
        if coincidencias >= 2:
            resultados[enfermedad] = coincidencias
    
    return max(resultados.items(), key=lambda x: x[1])[0] \
           if resultados else "No hay diagnóstico claro"

Integración con IA

Consultá a la IA: ¿Qué harías en mi lugar?

Los estudiantes pueden comparar sus sistemas de decisión con las recomendaciones de la IA:

Ejemplo de interacción:

# Escenario a consultar
situacion = "Tengo que elegir entre estudiar y salir con amigos"

# Prompt para el modelo
prompt = f"""
Tengo esta situación: "{situacion}"
¿Qué me recomendarías hacer y por qué?
"""

Proyecto integrador: "Tu propio asistente inteligente"

Objetivo

Construir un sistema en Python que tome decisiones por el usuario según variables como:

  • Estado de ánimo
  • Tiempo disponible
  • Objetivos personales
  • Contexto actual

Componentes sugeridos:

  • Interfaz con input() para recolectar información
  • Lógica condicional para procesar decisiones
  • Funciones modulares para cada tipo de decisión
  • Integración opcional con modelo de IA local
