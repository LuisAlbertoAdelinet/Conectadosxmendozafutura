Ejercicios Tecnológicos de IA en Arte y Creatividad

Una serie de actividades prácticas que combinan la inteligencia artificial con la expresión creativa, permitiendo a los estudiantes explorar las posibilidades artísticas de la IA mientras desarrollan habilidades técnicas y blandas.

Ejercicio 1: "Generación de Arte con IA y Python"

Objetivo: Crear imágenes artísticas usando una herramienta de IA y manipularlas con código Python.

Herramientas

  • Plataforma de IA gratuita (DeepDream Generator, DALL·E Mini, o Stable Diffusion via DreamStudio)
  • Librerías de Python: Pillow (PIL) para edición básica

Actividad

  1. Generación:

    Los estudiantes describen un concepto artístico (ej: "un bosque futurista con árboles de cristal") en una herramienta de IA para generar una imagen.

  2. Manipulación con Python:

    Usar Python para modificar la imagen generada (cambiar brillo, añadir filtros, recortar).

    from PIL import Image, ImageEnhance

img = Image.open("arte_ia.jpg")
enhancer = ImageEnhance.Brightness(img)
img_brighter = enhancer.enhance(1.5)  # Aumenta el brillo
img_brighter.save("arte_modificado.jpg")
  3. Reflexión:

    Discutir cómo la IA interpretó su idea y qué cambios aplicaron con código.

Ejercicio 2: "Música con IA y TensorFlow (Magenta)"

Objetivo: Componer una melodía simple usando IA y analizar el código subyacente.

Herramientas

  • Magenta Studio (biblioteca de TensorFlow para música)
  • Python Notebook en Google Colab

Actividad

  1. Generación de melodía:

    Usar un modelo preentrenado de Magenta (ej: MusicVAE) para generar notas musicales.

  2. Modificación:

    Cambiar tempo o instrumento en el código.

    # Ejemplo simplificado (usando Magenta en Colab)
from magenta.models.music_vae import configs
model = configs.CONFIG_MAP['cat-mel_2bar_big'].model
# ... (generar muestras y guardar como MIDI)
  3. Creatividad:

    Los estudiantes escuchan su melodía y proponen mejoras (ej: agregar percusión).

Ejercicio 3: "Poesía Colaborativa con IA"

Objetivo: Escribir un poema colaborativo humano-IA.

Herramientas

  • API de GPT-3 (via OpenAI Playground) o herramienta gratuita como InferKit

Actividad

  1. Inicio humano:

    Cada estudiante escribe 2 líneas de un poema.

  2. Contribución de la IA:

    Introducir las líneas en la herramienta de IA para que genere 2 líneas adicionales.

  3. Análisis:

    Comparar las contribuciones humanas vs. las de la IA: ¿Qué diferencias hay en el estilo?

Ejercicio 4: "Filtros de Arte con Redes Neuronales"

Objetivo: Aplicar filtros artísticos (estilo "neural style transfer") a fotos.

Herramientas

  • Librería tensorflow_hub en Google Colab

Actividad

  1. Transferencia de estilo:

    Usar un modelo preentrenado para combinar una foto común con un estilo artístico (ej: Van Gogh).

    import tensorflow_hub as hub
hub_model = hub.load('https://tfhub.dev/google/magenta/arbitrary-image-stylization-v1-256/2')
# ... (procesar imagen y estilo)
  2. Creatividad:

    Experimentar con diferentes estilos y discutir cómo la IA "entiende" el arte.

Ejercicio 5: "Reto de Diseño con Restricciones de IA"

Objetivo: Diseñar un póster bajo restricciones dadas por IA.

Herramientas

  • Canva + plugin de IA (ej: Text to Image)

Actividad

  1. La IA genera una consigna aleatoria (ej: "Crea un póster sobre el futuro con colores cálidos y un robot").
  2. Los estudiantes diseñan en Canva y votan el más creativo.

Habilidades Blandas Trabajadas

  • Creatividad: Adaptarse a las sugerencias de la IA.
  • Pensamiento crítico: Analizar resultados de la IA (¿son lógicos/artísticos?).
  • Trabajo en equipo: Ejercicios colaborativos (ej: poema o música).

Nota para el facilitador

  • Priorizar herramientas gratuitas y de bajo requerimiento técnico.
  • Usar ejemplos visuales o auditivos para mantener el engagement.
